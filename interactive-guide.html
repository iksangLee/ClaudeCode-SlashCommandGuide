🚀 Claude Code 내장 슬래시 명령어 가이드

대화형 세션에서 슬래시 명령어로 Claude의 동작을 제어하세요.

출처: Anthropic 공식 문서

📋 목차

📖 Claude Code 슬래시 명령어 개요

Claude Code는 21개의 강력한 빌트인 슬래시 명령어를 제공합니다. 이 가이드는 Anthropic 공식 문서를 기반으로 작성되었으며, 각 명령어의 정확한 기능과 사용법을 설명합니다.

🎯 빠른 시작 가이드
1단계: /help로 사용 가능한 명령어 확인
2단계: /status로 시스템 상태 점검
3단계: /model로 적합한 AI 모델 선택
4단계: /init로 프로젝트 초기화

💡 핵심 추천

가장 중요한 4개 명령어는 /help, /review, /status, /compact로, 이것들만 익혀도 Claude Code의 대부분 기능을 활용할 수 있습니다.

🎯 Part 1: 기본 시작 (Essential Start)

Claude Code를 처음 사용한다면 이 4개 명령어부터 시작하세요. 이 가이드는 Anthropic 공식 문서를 기반으로 작성되었으며, 각 명령어의 정확한 기능과 사용법을 설명합니다.

> /help
공식 기능: 사용법 도움말 보기
사용 시점: 명령어에 대해 알고 싶을 때

  • 모든 사용 가능한 슬래시 명령어 목록 표시
  • .claude/commands/의 프로젝트 커스텀 명령어 포함 (project 표시)
  • ~/.claude/commands/의 개인 커스텀 명령어 포함 (user 표시)
  • 연결된 MCP 서버의 명령어도 함께 표시
